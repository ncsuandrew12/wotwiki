# [Interview #991, Entry #3]: Brandon Sanderson's Blog: A MEMORY OF LIGHT ebook + Updates](https://www.theoryland.com/intvmain.php?i=991#3)

#### Brandon Sanderson

The annual Locus poll is live.
[Anyone can vote](http://www.locusmag.com/Magazine/2013/PollAndSurvey.html)
for what you consider to be 2012's best SF novel (among many other categories), but Locus subscribers' votes count double. The deadline is April 15th.

My agent Joshua Bilmes put up
[a blog post](http://brilligblogger.blogspot.com/2013/04/the-tachyons-soul.html)
about how
*The Emperor's Soul*
went from concept to Hugo nomination in 13 months.
